# ShopEase - E-Commerce Website

A responsive E-Commerce website developed using:

- Node.js
- Express.js
- EJS
- CSS
- MongoDB

## Features
✅ Home Page
✅ Product Listing
✅ Product Details
✅ Add to Cart
✅ Dynamic Shopping Cart
✅ Login and Registration
✅ Order Success Page
✅ Responsive Design
